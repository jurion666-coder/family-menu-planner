# 🍱 おうちの献立カレンダー (Family Menu Planner)

温かみのあるモダンなデザインと、家族の食卓を豊かにする便利なロジックを詰め込んだ、**「家族で共有できる1年間の献立カレンダー」**Webアプリケーションです。

インストールや難しいサーバー設定は不要で、ブラウザがあればスマホ・タブレット・PCのどれからでもすぐに利用でき、完全無料で家族向けにインターネット公開することも可能です。

---

## 🌟 わが家専用の「3つの献立ルール」を搭載！

本アプリには、ご要望いただいた以下の制約ルールがシステムとして組み込まれています。

1. **ハンバーグは日曜日限定！ 🍔**
   - 料理名に「ハンバーグ」が含まれるメニューを追加する際、カレンダーが**日曜日以外**の場合、かわいい警告メッセージが表示され保存が制限されます。「ハンバーグは日曜日のごちそう」ルールを完璧に守ります。
2. **同じ料理は最低1週間（7日間）空ける！ ⏳**
   - 「カレー」や「パスタ」など、料理が被ることは問題ありませんが、前後の1週間（登録日を中心に前後6日間・計13日間の範囲）にすでに同じ料理名が登録されている場合、重複を避けるための警告メッセージが表示され保存を制限します。これにより、献立のマンネリを防ぎ、レパートリー豊かな食卓をサポートします。
3. **1日あたり1品（主役メニュー） 🍳**
   - 1日につき1つのメイン料理をカレンダーにすっきりと美しく表示します。登録は「朝食・昼食・夕食・おやつ」のカテゴリーバッジから選ぶことができ、夕食がデフォルトになっています。

## ✨ プレミアムな機能一覧

* **極上のビジュアル & UI/UX**
  - 温かみのあるアーストーン（テラコッタ、セージグリーン、アンバー）を採用した、料理本のような心地よいデザイン。
  - 暗い部屋でも目に優しい**ダークモード**を搭載（太陽/月マークでいつでも切り替え可能）。
  - スマホでの使いやすさにこだわった**完全レスポンシブ対応**。スーパーでの買い物中に片手でサクサク確認できます。
* **1年間のカレンダー ↔ 月間詳細の切り替え**
  - **年間表示**: 12ヶ月分を一覧。献立がある日は可愛いドットで光り、全体のバランスが一目でわかります。月をクリックすると、ズームするように月間詳細に切り替わります。
  - **月間表示**: 大きなカレンダーでメニュー名や食事タイプ（夕食など）のバッジを表示します。
* **レシピURL連携 🔗**
  - 外部のレシピサイト（クックパッド、クラシル、デリッシュキッチンなど）のURLを登録でき、カレンダー上の「外部リンクボタン」からワンクリックでレシピページを開けます。
* **家族共有用のテキスト生成 💬**
  - 「今週」または「今月」の献立リストを、ワンクリックで綺麗なテキストとしてクリップボードにコピーできます。そのまま**LINEや家族チャットに貼り付けて一瞬で共有**できます！
* **自動買い物リスト生成 🛒**
  - 献立登録時のメモ欄に `- 牛肉 200g` や `- キャベツ` のように行頭に「- 」を付けて食材を書いておくと、**今週・今月の買い物リストに自動で食材を抽出・チェックリスト化**します！
  - リストには手動で「牛乳」や「日用品」などを追加・削除することもでき、スマホでチェックしながら買い物ができます。
* **データのバックアップと家族間同期 💾**
  - サーバー不要のLocalStorageを採用しているため、データはブラウザに安全に保存されます。
  - 「データエクスポート（ダウンロード）」と「インポート」機能を備えているため、家族の他のスマホにデータを送って同期したり、バックアップを取るのも簡単です。
  - 初めての方向けに、ルールに則った美味しい1週間分のメニューをワンクリックでセットできる**「デモデータ読み込み機能」**も搭載しています。

## 🚀 アプリの動かし方

このアプリは **HTML, CSS, JavaScript** のみで構築されているため、ビルドやインストール作業は一切不要です！

### 1. ローカルで動かす場合
1. `family-menu-planner` フォルダ内の `index.html` ファイルをダブルクリックしてブラウザで開くだけで動作します。

## 🌐 家族にWebアプリとして無料公開・共有する方法

カレンダーをインターネット上に公開し、家族全員がスマホからいつでもアクセスできるようにする最も簡単で安全な方法を3つご紹介します。すべて**完全無料**で利用できます。

### 方法A：Netlify（ネットリファイ）を使う方法 ★一番簡単・おすすめ！
1. [Netlify (https://www.netlify.com/)](https://www.netlify.com/) にアクセスし、無料アカウントを作成（またはログイン）します。
2. ログイン後、管理画面のメニューから **「Sites」** を開きます。
3. 画面下部にある **「Drag and drop your site folder here」** （ここにサイトのフォルダをドラッグ＆ドロップ）というエリアに、この `family-menu-planner` フォルダを丸ごとドラッグ＆ドロップします。
4. 数秒でアップロードが完了し、`https://xxxxxx.netlify.app` という家族専用の公開URLが自動生成されます！
5. 設定からURLを `https://our-family-menu.netlify.app` のように好きな名前に変更することも可能です。

### 方法B：Vercel（ヴァーセル）を使う方法
1. [Vercel (https://vercel.com/)](https://vercel.com/) にアクセスし、無料アカウントを作成します。
2. コマンドラインツールを使うか、GitHub連携をしてフォルダをアップロードするだけで、即座に高速なWebページとして公開されます。

### 方法C：GitHub Pages（ギットハブページズ）を使う方法（開発者向け）
1. このコードをGitHubのリポジトリにプッシュします。
2. リポジトリの **Settings > Pages** に移動します。
3. Build and deployment の Source で 「Deploy from a branch」 を選択し、`main` ブランチを指定して保存します。
4. 数分で `https://[ユーザー名].github.io/[リポジトリ名]/` にて世界中に公開されます。
